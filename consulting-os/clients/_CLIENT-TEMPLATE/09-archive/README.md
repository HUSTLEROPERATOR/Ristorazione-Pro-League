# 09 — Archivio

**Ristorante:** [NOME RISTORANTE]

Archivio di versioni precedenti, file storici e materiali non più attivi.

---

## Cosa archiviare qui

- Versioni precedenti di SOP o checklist (quando vengono aggiornate)
- Menu storici
- Report audit di versioni precedenti
- Comunicazioni rilevanti archiviate
- Qualsiasi file che non è più in uso ma vale la pena conservare

---

## Naming convention per i file archiviati

`[NOME-FILE-ORIGINALE]_v[N]_archiviato-[DATA].md`

Esempio: `sop-apertura_v1_archiviato-2025-06.md`
