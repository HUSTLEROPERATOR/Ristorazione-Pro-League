## Principio guida

Automatizza solo ciò che è già funzionante manualmente.
Prima testa il processo a mano, poi automatizzalo.

## Strumenti consigliati (no-code/low-cost)
- **Zapier / Make** — connessioni tra app
- **Brevo** — email marketing + WhatsApp (piano free)
- **Google Forms + Sheets** — raccolta dati
- **Calendly** — prenotazioni semplici
