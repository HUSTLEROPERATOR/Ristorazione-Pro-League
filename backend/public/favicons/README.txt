How to verify locally:
1. Start the simplified server in backend:
   npm run simple
2. Open in browser: http://localhost:4000
3. Or check favicon directly: http://localhost:4000/favicons/01U8vspIR4mlslDQvPXMFA-32.png

If server does not respond:
- Ensure no other node processes are running: taskkill /F /IM node.exe
- Start server again: npm run simple
- If port 4000 is blocked, change port in src/simple-server.ts (PORT variable) and restart.
